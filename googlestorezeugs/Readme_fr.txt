Pour les developeurs, developant directements sous Android, les environnments de developpement, tels qu'AIDE, ne gerent pas les ressources string, cette application permet de gerer les ressource de chaines de caracteres en forme tabulaire

Cette application est a utiliser en conjonction avec une IDE comme AIDE, on ne peut creer de nouveaux projets, c'est le travail de l'IDE...

Aide pour debuter:

- cliquer sur le chemin pour ouvrir un dialogue permettant de selectionner le projet a travailler, selectionner le noeud racine du projet suffit, s'il y avait des ambiguites un dialogue s'ouvrira pour les resoudre.
- la premiere colonne represente le mot clef, en cliquant sur l'entete de cette colonne on peut ouvrir un dialogue pour rajouter un mot clef, renseignez le type le nom et une valeur par defaut,  en cliquant sur un mot clef, un menu de contexte s'ouvrira, pour p.ex. l'effacer (pas de retour possible)
-en cliquant sur l'entete d'une colonne de traductions on la cache
- en cliquant sur l'entete de coonne appele "..." l'on peut rendre les colonnes cachees visibles a nouveau ou rajouter une langue
- en cliquant dans un champ avecc   une ttraduction on peut la modifier, attention a cec que vous tapez, aucune anaylse de contenu n'est faite, donc evitez les signes actifs du xml ou sachiez ce que vous faites.
A la fin, tapez "entree" pour valider la modification, sinon elle sera perdu (meme si visible)

et n'oubliqez pas de clicquer sur la disquette pour sauvegarder le tout!
